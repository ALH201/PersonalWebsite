Course Information

During my time at Wayne State University, I was able to take a variety of courses that covered different parts in technology.
The knowledge I've obtained will continue to help me as I progress within my own career.

Why don't we take a closer look at my favorites?

Course Description
System Administration Learned how virtualization works, how to implement and manipulate processes, install and configure Linux software packages, understand Linux user management, grasped how permissions work, write bash scripts, understood how the network communicates between servers, utilized logging and monitoring to detect potential threats, and understanding docker.
Cyber Security Practice This course allows us to explore board security topics in areas of network and operating systems. The Linux distribution we're using is Ubuntu. After creating the necessary containers, we see how threats are carried out by launching them and creating defensive mechanisms against these attacks.
Database Systems This course allowed us to see the importance of data within today's enterprise. It covered various theories, models, and techniques for designing, developing, and creating and manipulating a database. I got a chance to work on a relational database, while also learning data modeling, physical database design, database implementation, and a bit of SQL.
Computer Networking and Applications This course gave a full overview of computer networking. A top-to-bottom approach was leveraged so I could understand essential network concepts and protocols, from high-level applications to low-level operational functions, including HTTP, TCP/IP, routing LAN, wireless, etc. I also got a chance to do some socket programming in Python for data communications.
Principles of Web Design In this course, I learned new skills for building websites. It covered the fundamentals of HTML, CSS, Apache, MySQL, JavaScript, and PHP. It gave me also gave me chance to become familiar with the Document Object Model and asynchronous JavaScript and XML.
Senior Design Project For my Senior Project I decided to create a Stable Diffusion Application on Wayne State University's HPC grid, which can be accessed through their Open Ondemand portal. This is implemented on the Centos 7 Linux distribution..
Coding Languages Python, Bash, SQL, JavaScript, HTML/CSS, Bootstrap, Java, C++, and C
